JMkvpropedit v1.0.9.1

A batch GUI for mkvpropedit (part of MKVToolNix) written in Java.
It should work on Windows, Linux and other *nixes (not tested).
Requires Java 6 or newer.

Source and downloads at:
http://jmkvpropedit.googlecode.com/

---

Regarding the "Match file" option of chapters and tags, it works this way:
File: example.mkv
Chapters file: example.xml
Tags file: example-tags.xml
